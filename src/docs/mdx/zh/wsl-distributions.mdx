## WSL Linux 分发版 镜像使用帮助

## 地址

https://mirrors.cqu.edu.cn/github-release/microsoft/WSL/

## 说明

有关WSL的简介和安装说明，请访问[WSL 镜像使用帮助](/mirrors/github-release)。

## 收录版本
- Arch Linux
- Fedora
- Ubuntu

其余未收录的发行版请访问官方[DistributionInfo.json](https://github.com/microsoft/WSL/blob/master/distributions/DistributionInfo.json)获取下载链接。

## 使用说明

请访问[主页](/)右侧"获取下载链接"-"系统镜像"-"WSL Linux 分发版"并选择合适版本下载。
