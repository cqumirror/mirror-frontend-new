## Maven 反向代理使用帮助

## 地址

https://mirrors.cqu.edu.cn/maven

## 说明

Maven 是一个流行的 Java 项目管理和构建工具，广泛用于 Java 项目的依赖管理和构建过程。

本站提供 Maven Central、Gradle 以及其他一些常用的 Maven 仓库的反向代理服务。



## 使用说明

### 反向代理说明

### Maven 配置
以 Maven 中央仓库为例，以下是一个示例配置：

#### 全局配置

- linux 用户在终端中输入以下命令:
```bash
mkdir ~/.m2
vim ~/.m2/settings.xml
```
- windows 用户在 powershell (或者pwsh中) 输入以下命令：
```powershell
mkdir ~/.m2
cd ~/.m2
notepad settings.xml
```

如果打开的文件为空，粘贴以下内容：
```xml
<?xml version="1.0" encoding="UTF-8"?>
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0 http://maven.apache.org/xsd/settings-1.0.0.xsd">
    <mirrors>
       <mirror>
            <id>cqu_mirror</id>
            <mirrorOf>*</mirrorOf>
            <url>https://mirrors.cqu.edu.cn/maven/</url>
        </mirror>
    </mirrors>
</settings>
```
如果已经有内容，则在 mirrors 节点加入子节点 (对于 mirrorOf 值相同的节点，只有首个子节点生效)：
```xml
<mirror>
    <id>cqu_mirror</id>
    <mirrorOf>*</mirrorOf>
    <url>https://mirrors.cqu.edu.cn/maven/</url>
</mirror>
```

#### 项目配置
如果只在某个项目中使用，则在项目的 `pom.xml` 中配置：
```xml
<project>
    ......[其他配置]

    <repositories>
        <repository>
            <id>cqu</id>
            <url>https://mirrors.cqu.edu.cn/maven/</url>
            <releases>
                <enabled>true</enabled>
            </releases>
            <snapshots>
                <enabled>false</enabled>
            </snapshots>
        </repository>
    </repositories>
</project>
```
