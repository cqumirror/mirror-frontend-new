## Arch Linux Mirror Usage Guide

## Address

https://mirrors.cqu.edu.cn/archlinux/

## Description

Arch Linux package mirror

## Supported Architectures

x86_64

## Usage

As root, manually edit the file `/etc/pacman.d/mirrorlist` and add the following line at the top of the file:

```bash
Server = https://mirrors.cqu.edu.cn/archlinux/$repo/os/$arch
```
