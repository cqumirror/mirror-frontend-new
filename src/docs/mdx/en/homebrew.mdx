## Homebrew Mirror Usage Guide

## URL

https://mirrors.cqu.edu.cn/homebrew/

## Description

The Homebrew mirror consists of two parts: the formula index source and the bottle source containing pre-built binary files. This mirror is for the Homebrew formula index, corresponding to `brew update`, not `brew upgrade` or `brew install`.


Configuring this source will only improve the speed of `brew update`. To speed up `brew upgrade` and `brew install`, please refer to the [Homebrew-bottles Mirror Usage Guide](/mirrors/homebrew-bottles) to configure the bottle source.


## Usage
### Initial Installation
First, make sure bash, git, and curl are installed on your system. macOS users additionally need to install Command Line Tools (CLT) for Xcode.
- For macOS users, bash, git, and curl are pre-installed. Simply run `xcode-select --install` in the terminal to install CLT for Xcode.
- For Linux users, bash is pre-installed. You only need to install git and curl.

Next, enter the following lines in the terminal to set environment variables:
```sh
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.cqu.edu.cn/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.cqu.edu.cn/homebrew/homebrew-core.git"
export HOMEBREW_INSTALL_FROM_API=1
# export HOMEBREW_API_DOMAIN
# export HOMEBREW_BOTTLE_DOMAIN
# export HOMEBREW_PIP_INDEX_URL
```

It is recommended to also refer to the [Homebrew-bottles Mirror Usage Guide](/mirrors/homebrew-bottles) to set `HOMEBREW_API_DOMAIN` and `HOMEBREW_BOTTLE_DOMAIN`, and the [PyPI Mirror Usage Guide](/mirrors/pypi) to set `HOMEBREW_PIP_INDEX_URL`.

Finally, run the following commands in the terminal to install:
```shell
# Download the installation script from the mirror and install Homebrew / Linuxbrew
git clone --depth=1 https://mirrors.cqu.edu.cn/homebrew/install.git brew-install
/bin/bash brew-install/install.sh
rm -rf brew-install

# Alternatively, get the official installation script from GitHub to install Homebrew / Linuxbrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```


### Replacing Existing Repository Upstream
 Add the following to your shell configuration file, such as `.zshrc` or `.bash_profile`:

 ```sh
 export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.cqu.edu.cn/git/homebrew/brew.git"
 export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.cqu.edu.cn/git/homebrew/homebrew-core.git"
 ```

 Then source the corresponding shell configuration file and run `brew update` to update the Homebrew formula index.

 After the update completes, run `brew config` and check the `HOMEBREW_BREW_GIT_REMOTE` and `HOMEBREW_CORE_GIT_REMOTE` entries — they should now point to this mirror.
