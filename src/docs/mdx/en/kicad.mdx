## Description
### Introduction to KiCad ([from Wikipedia](https://en.wikipedia.org/wiki/KiCad))
KiCad (pronounced key-CAD) is a cross-platform open-source Electronic Design Automation (EDA) suite.
Originally released in 1992 by Frenchman Jean-Pierre Charras. The suite provides tools for schematic design, PCB layout, symbol library design, footprint library design, 3D PCB visualization, Gerber viewing, and PCB utility calculations. It supports 22 languages including Chinese. In 2013, CERN's (European Organization for Nuclear Research) BE-CO-HT department began contributing resources to support it as a tool suite comparable to commercial EDA tools in the open-source hardware field. KiCad is currently developed and maintained by the KiCad team.

### KiCad Directory Description

| docs | KiCad manuals | Yes |
| doxygen | KiCad development configuration | Yes |
| doxygen-python | KiCad Python development configuration | Yes |
| libraries | KiCad footprint libraries | Yes |
| osx | KiCad packages for Mac OS | Yes |
| windows | KiCad packages for Windows | Yes |

| favicon.ico | Icon | No |
