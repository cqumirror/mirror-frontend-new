## Blender Mirror Usage Guide

## Address

http://mirrors.cqu.edu.cn/blender/

## Description
### Blender Introduction ([from Wikipedia](https://en.wikipedia.org/wiki/Blender))
Blender is a free and open-source 3D computer graphics software that can be used to create and render 3D models, animations, games, visual effects, and more.

## Supported Architectures
- x86_64
- arm64

## Usage

Only installation packages are synced; compressed archives are not provided. If needed, please download them from the official website.

Please visit the [homepage](/) and on the right side click "Get Download Links" - "Common Software" - "Blender" and select the appropriate version to download.

## Supported Versions
Latest version
