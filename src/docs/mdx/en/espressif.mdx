## Espressif Mirror Usage Guide

## URL

https://mirrors.cqu.edu.cn/espressif

## Description

Espressif Systems is a leading global fabless semiconductor company. Its ESP series products are widely used and highly regarded.

This repository mirrors a selection of Espressif open-source projects.

This mirror is currently in trial operation. Please report any issues promptly.

## Mirrored Repositories

Please visit https://mirrors.cqu.edu.cn/git/espressif/ for the latest repository list.

Since Espressif has many projects with frequent updates, we selectively mirror repositories based on user demand and project importance. If you need a specific repository, please let us know.

## Usage

### Initial Clone

For the initial clone, it is recommended to use the [sync script library](https://mirrors.cqu.edu.cn/git/espressif/scripts/) provided by the mirror. It will automatically clone recursively and handle submodule URLs, so that all fetching is done through the mirror.

For example, to clone the esp-idf repository, use the following command:

```bash
bash <(curl -s https://mirrors.cqu.edu.cn/git/espressif/scripts/esp-idf.sh)
```

### Updating Repositories

If you used the sync script library for the initial clone, the remote URLs of all repositories have already been replaced with the mirror address. For subsequent updates, simply use `git pull` or other fetch commands with no additional steps required.
