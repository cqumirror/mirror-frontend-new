## Manjaro Linux Mirror Usage Guide

## URL
https://mirrors.cqu.edu.cn/manjaro/

## Description
Manjaro Linux software repository.

## Supported Architectures
- x86_64
- i686

## Usage
Manually edit the file `/etc/pacman.d/mirrorlist` as root, and add the following line at the top of the file:
```bash
Server = https://mirrors.cqu.edu.cn/manjaro/stable/$repo/$arch
```
