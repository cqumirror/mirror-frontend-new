## Description
### BMCLAPI Introduction ([from the official documentation](https://bmclapidoc.bangbang93.com))
BMCLAPI is a part of BMCL developed by @bangbang93, designed to solve the slow speed issue with Forge and Minecraft's official Amazon S3 in China's network. BMCLAPI is open to the public and can be called by any launcher that needs Minecraft resources.

## Usage

Only partial open-source project mirroring is provided, syncing files from Forge, NeoForge, and Maven Center. Private files are not included.

This mirror site has been added to the BMCLAPI mirror list. When using the official API service, requests will be automatically redirected to this site based on proximity, so no manual configuration is needed.
