## Gentoo-zh Community Overlay Mirror Usage Guide

## URL

http://mirrors.cqu.edu.cn/git/gentoo-zh.git

## Description

Gentoo zh Overlay

## Supported Architectures

- Defined in ebuild

## Usage
### Initial Setup


Make sure `git` is installed beforehand. Without `git`, all of the following index and add steps will fail.


Ensure `git` is installed. If not, install it with:

```bash
sudo emerge -aq git
```

If you have previously synced `gentoo-zh`, delete the existing content and re-sync:

```bash
rm -rf /var/db/repos/gentoo-zh
```

Run the following to add the gentoo-zh mirror:


```bash
eselect repository add gentoo-zh git https://mirrors.cqu.edu.cn/git/gentoo-zh.git
```

Then run the sync:

```bash
emerge --sync gentoo-zh
```

Search for packages with eix:

```bash
eix <package>
```

Install gentoo-zh packages:

```bash
emerge -aq <package>
```


gentoo-zh is a complete mirror of https://github.com/microcai/gentoo-zh. It is an [ebuild](https://wiki.gentoo.org/wiki/Ebuild) repository and does not contain the binary and source code content referenced by the ebuilds.


If you need to use the `distfiles cache`, please refer to the instructions in [Gentoo zh distfile Cache Mirror Usage Guide](/mirrors/gentoo-zh).


Some packages in gentoo-zh require enabling the `~amd64` keyword. If stability is a priority, please weigh this accordingly.
