## WSL Linux Distributions Mirror Usage Guide

## URL

https://mirrors.cqu.edu.cn/github-release/microsoft/WSL/

## Description

For an introduction to WSL and installation instructions, please visit [WSL Mirror Usage Guide](/mirrors/github-release).

## Supported Versions
- Arch Linux
- Fedora
- Ubuntu

For distributions not listed above, please visit the official [DistributionInfo.json](https://github.com/microsoft/WSL/blob/master/distributions/DistributionInfo.json) to obtain download links.

## Usage

Please visit the [homepage](/) right side "Get Download Links" - "System Images" - "WSL Linux Distributions" and select the appropriate version to download.
