## CentOS Mirror Usage Guide

As CentOS has reached end of life, this mirror's CentOS repository only serves on-campus users and only provides mirror service for version [7.9.2009](https://mirrors.cqu.edu.cn/centos/7.9.2009/).

## Address

http://mirrors.cqu.edu.cn/CentOS

## Description

CentOS package source

## Supported Architectures

- x86_64
- i386

## Supported Versions

- CentOS 7.9.2009

## Usage

First, back up `CentOS-Base.repo`:

```bash
mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup
```

Download the file to the specified directory:

```bash
wget   -O   /etc/yum.repos.d/CentOS-Base.repo  http://mirrors.cqu.edu.cn/repo/centos/Centos-7.repo
```

In some cases, `wget` may not be pre-installed on the system. Please install it in advance.

Then refresh the yum cache:

```bash
yum makecache
```
