## Homebrew Bottles Mirror Usage Guide

## URL

https://mirrors.cqu.edu.cn/homebrew-bottles

## Description
This is a mirror for Homebrew pre-built binary packages (bottles). The mirror also provides a mirror for Homebrew's formula index (i.e., the content updated during `brew update`). Please refer to the [Homebrew Mirror Usage Guide](/mirrors/homebrew).

(Linuxbrew users) The Linuxbrew core repository (Linuxbrew/brew, version >= 3.3.0) was deprecated on October 25, 2021. Linuxbrew users should migrate to homebrew-core. Linuxbrew users should follow the updated instructions for mirror configuration.

## Usage

Set the environment variable `HOMEBREW_BOTTLE_DOMAIN` to `https://mirrors.cqu.edu.cn/homebrew-bottles` before running `brew`.
Additionally, brew 4.0 and later versions use a new metadata JSON API, so you also need to set the environment variable `HOMEBREW_API_DOMAIN` to `https://mirrors.cqu.edu.cn/homebrew-bottles/api`.

### Temporary Replacement

Set the environment variables in the current terminal session. This only takes effect for the current session.

```sh
export HOMEBREW_API_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles/api/"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles"
```

### Permanent Replacement
Add the environment variables to your shell configuration file to make them persistent.

```sh
# For Bash
echo 'export HOMEBREW_API_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles/api/"' >> ~/.bash_profile
echo 'export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles"' >> ~/.bash_profile
export HOMEBREW_API_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles/api/"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles"
# For Zsh
echo 'export HOMEBREW_API_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles/api/"' >> ~/.zprofile
echo 'export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles"' >> ~/.zprofile
export HOMEBREW_API_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles/api/"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.cqu.edu.cn/homebrew-bottles"
```
