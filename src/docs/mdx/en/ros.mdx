## ROS Mirror Usage Guide

## URL

https://mirrors.cqu.edu.cn/ros/ubuntu

## Description


ROS 1 software repository

## Applicable Systems

* Ubuntu 20.04 LTS
* Ubuntu 18.04 LTS
* Ubuntu 16.04 LTS
* Ubuntu 14.04 LTS
* Debian 10
* Debian 9
* Debian 8


> For Ubuntu 22.04 LTS and above, please use ROS 2, or install ROS 1 using Docker.


## Usage


### Install Required Packages

```bash
sudo apt update && sudo apt install gpg
```

### Add the ROS 1 GPG Key

Download the PGP Key and import it with gpg

```bash
gpg --keyserver hkp://keyserver.ubuntu.com --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654
```

If your device can only access the campus network and cannot reach the internet, you can use the following command to download and import the OpenPGP public key from the CQU mirror (not recommended for other cases):

```bash
gpg --keyserver hkps://mirrors.cqu.edu.cn  --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654
```

Export the GPG public key file

```bash
gpg --export C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654 | sudo tee /etc/apt/trusted.gpg.d/ros1-latest.gpg > /dev/null
```

(Optional) Remove the ROS 1 public key from the gpg keyring

```bash
gpg --delete-keys C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654
```

### Set Up the ROS 1 Mirror Source:

```bash
echo "deb [signed-by=/etc/apt/trusted.gpg.d/ros1-latest.gpg] http://mirrors.cqu.edu.cn/ros/ubuntu $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/ros1-latest.list > /dev/null

sudo apt update
```
