## OpenWrt / LEDE Mirror Usage Guide

## URL
http://mirrors.cqu.edu.cn/openwrt/

## Description
OpenWrt / LEDE software repository.

## Supported Architectures
All officially supported architectures.

## Usage
### Luci
In Luci, navigate to: `System` -> `Software` -> `Advanced`
Find the software source editing area, and replace all occurrences of `downloads.openwrt.org` with `mirrors.cqu.edu.cn/openwrt`.
Click `Submit` to apply the changes.
Switch to the `Package` page and click `refresh` to update the package manager cache.

### Shell

Please make sure to change the root password promptly after logging in.

Use an OpenSSH-compatible tool such as PuTTY. On Linux and macOS, you can execute directly in the terminal:
```bash
ssh root@IP -p PORT
# Replace IP and PORT with actual values; the default port is usually 22
```
Then run the following command:
```bash
cp /etc/opkg/distfeeds.conf /etc/opkg/distfeeds.conf.save
```
to back up the default software source list.
Then run:
```bash
sed -i 's/downloads.openwrt.org/mirrors.cqu.edu.cn\/openwrt/g' /etc/opkg/distfeeds.conf
opkg update
```
to change the software source and refresh the cache.

Please ensure you have a working network connection when refreshing the cache. If you are on the Chongqing University campus, please make sure you have a valid campus network connection (no login authentication required).
