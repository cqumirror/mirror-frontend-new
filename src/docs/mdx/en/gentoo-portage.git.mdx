## Gentoo Portage Git Mirror Usage Guide

## URL

http://mirrors.cqu.edu.cn/git/gentoo-portage.git

## Description

Gentoo portage git sync repository.

## Supported Architectures

- Defined in ebuild

## Usage
### Initial Setup

When switching to the **git** sync method for the first time, you need to clear the previously synced content:

```bash
rm -rf /var/db/repos/gentoo
```

Then edit `/etc/portage/repos.conf/gentoo.conf` with the following changes:

```ini
# change sync-type to git
sync-type=git
```

```ini
# change sync-uri
sync-uri=https://mirrors.cqu.edu.cn/git/gentoo-portage.git
```

Then run the sync:

```bash
emerge --sync
```


The initial sync will take some time.
