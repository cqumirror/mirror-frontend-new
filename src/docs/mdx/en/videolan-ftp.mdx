## Description
### VLC media player Introduction<sup><a href="https://www.videolan.org/vlc/index.zh_CN.html">From the official website</a></sup>
VLC is a free and open source cross-platform multimedia player and framework that plays most multimedia files as well as DVDs, Audio CDs, VCDs, and various streaming protocols.

## Supported Architectures
- universal
- x86
- x86_64
- arm64

## Supported Versions
Latest version

## Usage

Only desktop platform and Android mobile installation packages are synchronized.

Please visit the [homepage](/) right side "Get Download Links" - "Common Software" - "VLC" and select the appropriate version to download.
