## GitHub Release Mirror Usage Guide

## URL
https://mirrors.cqu.edu.cn/github-release

## Description
Mirrors select software release packages and font files from GitHub Releases.

Please note that the mirror site cannot perform security audits on all software. The mirror site only provides download services; please assess the safety of the software yourself and assume the corresponding risks.
