## Maven Reverse Proxy Usage Guide

## URL

https://mirrors.cqu.edu.cn/maven

## Description

Maven is a popular Java project management and build tool, widely used for dependency management and build processes in Java projects.

This site provides reverse proxy services for Maven Central, Gradle, and several other commonly used Maven repositories.

## Usage

### Maven Configuration
Using Maven Central as an example, here is a sample configuration:

#### Global Configuration

- Linux users, enter the following commands in the terminal:
```bash
mkdir ~/.m2
vim ~/.m2/settings.xml
```
- Windows users, enter the following commands in PowerShell (or pwsh):
```powershell
mkdir ~/.m2
cd ~/.m2
notepad settings.xml
```

If the opened file is empty, paste the following content:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0 http://maven.apache.org/xsd/settings-1.0.0.xsd">
    <mirrors>
       <mirror>
            <id>cqu_mirror</id>
            <mirrorOf>*</mirrorOf>
            <url>https://mirrors.cqu.edu.cn/maven/</url>
        </mirror>
    </mirrors>
</settings>
```
If the file already has content, add the following child node inside the `<mirrors>` node (for nodes with the same `mirrorOf` value, only the first child node takes effect):
```xml
<mirror>
    <id>cqu_mirror</id>
    <mirrorOf>*</mirrorOf>
    <url>https://mirrors.cqu.edu.cn/maven/</url>
</mirror>
```

#### Project Configuration
If you only want to use it for a specific project, configure it in the project's `pom.xml`:
```xml
<project>
    ......[other configurations]

    <repositories>
        <repository>
            <id>cqu</id>
            <url>https://mirrors.cqu.edu.cn/maven/</url>
            <releases>
                <enabled>true</enabled>
            </releases>
            <snapshots>
                <enabled>false</enabled>
            </snapshots>
        </repository>
    </repositories>
</project>
```
