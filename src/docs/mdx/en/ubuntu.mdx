## Ubuntu Linux Mirror Usage Guide

## URL

https://mirrors.cqu.edu.cn/ubuntu/

## Description

Ubuntu software repository

## Supported Architectures

x86_64

## Supported Versions

Release

For details, please refer to [Ubuntu Wiki](http://wiki.ubuntu.com/Releases)

## Usage

### Simple Method


This method is only applicable to the Desktop edition.


1. Click the menu on the `left dock` to enter the `Gnome menu`

2. Search for `software` and open the `Software & Updates` application

3. In the dialog that opens, use the dropdown menu on the **first tab** (`Download from`) to select `Other Sites`. In the new dialog, select the China list and choose `mirrors.cqu.edu.cn`, then click `Choose Server`

4. Click Close. You will be prompted to refresh the apt cache. Select **Yes** to refresh the apt cache, and it will be ready to use

### General Method


Before **Ubuntu 24.04**, Ubuntu's software source configuration file used the traditional **One-Line-Style** format, located at `/etc/apt/sources.list`. Starting from **Ubuntu 24.04**, Ubuntu's software source configuration file has changed to the [DEB822 format](#deb822-format), located at `/etc/apt/sources.list.d/ubuntu.sources`.


The following commands are for versions **before Ubuntu 24.04**. For **Ubuntu 24.04**, `/etc/apt/sources.list` will be an **empty file** and needs to be **filled directly** with the appropriate software source configuration, while **deleting** `/etc/apt/sources.list.d/ubuntu.sources` to prevent duplicate configuration. Therefore, it is recommended to **use the [DEB822 format](#deb822-format) directly**.



First, back up the `sources.list` file

```bash
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
```

Then run

```bash
sudo nano /etc/apt/sources.list
```

Use `ctrl` + `\` to search for `archive.ubuntu.com`, and replace it with `mirrors.cqu.edu.cn`, press Enter to submit, then press `shift` + `a` (i.e., `A`) to replace all occurrences.
Press `Ctrl` + `o` to write changes, then press `ctrl` + `x` to exit.

> If the source address is `cn.archive.ubuntu.com`, please replace `cn.archive.ubuntu.com` with `mirrors.cqu.edu.cn`

The result should typically look like this:

```bash

# deb-src http://mirrors.cqu.edu.cn/ubuntu/ bionic main restricted universe multiverse
```

### DEB822 Format

The following commands are only for **Ubuntu 24.04 and later** versions.

First, back up the `ubuntu.sources` file

```bash
sudo cp /etc/apt/sources.list.d/ubuntu.sources /etc/apt/sources.list.d/ubuntu.sources.bak
```

Then run

```bash
sudo nano /etc/apt/sources.list.d/ubuntu.sources
```

Replace its contents with:

For Ubuntu 24.04 (noble)

```bash
Types: deb
URIs: https://mirrors.cqu.edu.cn/ubuntu
Suites: noble noble-updates noble-backports
Components: main restricted universe multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

# Source mirrors are commented out by default to speed up apt update; uncomment if needed
# Types: deb-src
# URIs: https://mirrors.cqu.edu.cn/ubuntu
# Suites: noble noble-updates noble-backports
# Components: main restricted universe multiverse
# Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

# The following security update source includes both official and mirror configurations; modify as needed
# Types: deb
# URIs: https://mirrors.cqu.edu.cn/ubuntu
# Suites: noble-security
# Components: main restricted universe multiverse
# Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

# # Types: deb-src
# # URIs: https://mirrors.cqu.edu.cn/ubuntu
# # Suites: noble-security
# # Components: main restricted universe multiverse
# # Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

Types: deb
URIs: http://security.ubuntu.com/ubuntu/
Suites: noble-security
Components: main restricted universe multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

# Types: deb-src
# URIs: http://security.ubuntu.com/ubuntu/
# Suites: noble-security
# Components: main restricted universe multiverse
# Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

# Pre-release software source, not recommended to enable
# Types: deb
# URIs: https://mirrors.cqu.edu.cn/ubuntu
# Suites: noble-proposed
# Components: main restricted universe multiverse
# Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

# # Types: deb-src
# # URIs: https://mirrors.cqu.edu.cn/ubuntu
# # Suites: noble-proposed
# # Components: main restricted universe multiverse
# # Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg
```

Press `Ctrl` + `o` to write changes (press `Enter` to confirm the filename), then press `ctrl` + `x` to exit.

Then run

```bash
sudo apt update
```

Refresh the apt cache to apply changes.

Run

```bash
sudo apt upgrade
```
