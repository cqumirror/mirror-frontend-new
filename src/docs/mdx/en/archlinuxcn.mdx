## Arch Linux CN Mirror Usage Guide

## Address

https://mirrors.cqu.edu.cn/archlinuxcn/

## Description

Arch Linux CN Repository

## Introduction

The Arch Linux Chinese Community Repository is an unofficial user repository driven by the Arch Linux Chinese community. It contains software, tools, fonts, and theme packages commonly used by Chinese users.

## Supported Architectures

- x86_64
- arm64

## Usage

As root, manually edit the file `/etc/pacman.conf` and add the following lines at the end of the file:

```bash
[archlinuxcn]
Server = https://mirrors.cqu.edu.cn/archlinuxcn/$arch
```

Run:

```bash
pacman -Syy
```

to force a refresh of the package sources. Then run:

```bash
pacman -S archlinuxcn-keyring
```

to add the Arch Linux CN GPG key.

It is recommended to set `SigLevel` to `TrustedOnly`.
